## 背景介绍
一般而言，对于任意形状的台球桌，台球的运动轨迹是混沌的，但是如果台球桌具有一定的对称性，台球的轨迹应该大体上有规律可循。考虑椭圆形的桌子，球从椭圆的焦点出发。

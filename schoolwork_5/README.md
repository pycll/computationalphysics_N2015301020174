## 题目大意
教材上的习题2.21，计算一个以2000rpm侧旋击出的棒球的运动轨迹，并计算棒球的偏移量。

## 我的想法
在考虑空气粘性的情况下，由于棒球在飞行中不断地旋转，由于马格努斯效应会偏离原来的飞行轨迹。具体而言，棒球受到了一个马格努斯力，其正比于旋转棒球两侧的气体流速的平方差，而且方向随着棒球的运动不断地变化。
